# Django

```
django-admin startproject [name]      # Creates a new project

python manage.py runserver            # Starts development server at port 8000
python manage.py startapp [name]      # Start new app


python manage.py migrate              # Creates/Edits tables in db
python manage.py makemigrations [name]# Where name is app config
python manage.py sqlmigrate [app] [migrateID] # View corresponding sql commands

python manage.py createsuperuser      # Creates superuser 
```
